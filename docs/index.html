Restaurant Management API

A comprehensive backend solution for restaurant operations

About This Project

This is a full-featured Restaurant Management System API built with Go and Gin framework. The system provides a complete backend solution for managing restaurant operations including user authentication, menu management, food ordering, and invoice generation.

Key Features

Authentication

Secure user authentication with JWT tokens and bcrypt password hashing for robust security

Food Management

Complete CRUD operations for food items with menu category organization and pricing

Order Processing

Handle customer orders and order items with real-time status tracking and updates

Table Management

Organize and manage restaurant tables with seating capacity and availability status

Invoice Generation

Automated invoice creation and management with detailed order summaries and totals

MongoDB

Powered by MongoDB for scalable data storage with flexible schema and performance

Technology Stack

  • Language: Go (Golang)
  • Framework: Gin Web Framework
  • Database: MongoDB
  • Authentication: JWT (JSON Web Tokens)
  • Documentation: Swagger/OpenAPI
